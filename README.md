# Forum ALTLINK

## Запуск

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Локал `http://127.0.0.1:5000`.

```powershell
.\.venv\Scripts\python.exe app.py
```

## Что имеется

- Главная страница форума на `Flask`
- Регистрация аккаунта с `nickname`, паролем, подтверждением пароля, полом и загрузкой аватарки
- Отдельная страница профиля пользователя
- ORM-модель `User` через `Flask-SQLAlchemy`
- Работа с контекстом текущего пользователя через сессии Flask и `g.current_user`
- Темы форума и сообщения пользователей внутри тем
- Загрузка аватарки пользователя (только изображения)
- Локальная база `SQLite`
- Дефолтная аватарка из папки `media`, если пользователь не загрузил свою
- Автоответ Поддержки ALTLINK через `Gemini` в теме `Проблемы и решения`
- Нет поддержки загрузки музыки или обработки медиафайлов как контекста для Gemini

## Технологии

- `Flask` — веб-фреймворк для маршрутов, сессий и шаблонов.
- `Flask-SQLAlchemy` — ORM для работы с SQLite.
- `SQLite` — локальная база данных для пользователей, тем и сообщений.
- `Jinja2` — шаблонизатор HTML.
- `requests` — HTTP-клиент для обращения к Gemini API.

## Настройка Gemini

В файле `.env` используется переменная:

```env
api_gemini_url=
```

Базовый промтик для нейронки хранится в [gemini_prompt.json]
